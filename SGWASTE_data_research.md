## 1. Singapore Recycling Rates by Year

### Overall Recycling Rate (includes industrial + construction waste)

| Year | Overall Rate | Notes |
|------|-------------|-------|
| 2015 | 61% | Stable plateau |
| 2016 | 61% | — |
| 2017 | 61% | — |
| 2018 | 62% | Peak year |
| 2019 | 59% | Slight dip |
| 2020 | 52% | COVID-19 impact |
| 2021 | 55% | Partial recovery |
| 2022 | 57% | — |
| 2023 | 52% | Drop — less C&D and slag waste |
| 2024 | 50% | New decade low |

> ⚠️ **Critical context for the site:** The 50%+ figure is heavily skewed by industrial, construction & demolition (C&D), and used slag waste — which are almost fully recycled. This makes the headline number look much better than the reality.

### Domestic (Household) Recycling Rate — the REAL number for everyday people

| Year | Domestic Rate |
|------|--------------|
| 2022 | 12% |
| 2023 | 12% |
| 2024 | **11%** — new all-time low |

**Target under Singapore Green Plan 2030:** 30% domestic recycling rate by 2030.
**Gap:** Singapore needs to nearly triple its household recycling rate in 6 years.

### Blue Bin Contamination
- ~**40%** of what Singaporeans put in blue recycling bins cannot actually be recycled — contaminated with food/liquid, or wrong materials entirely (soft toys, clothing)
- This means: even when people try to recycle, nearly half of it ends up incinerated anyway

### Plastic-Specific Recycling Rate
- Singapore's plastic recycling rate: **5%** (2024)
- One of the lowest recyclable material rates

## 2. Total Waste Generated

- **2024:** ~6.66 million tonnes solid waste generated; ~3.33 million tonnes recycled (50%)
- Waste generation per capita has been trending downward — positive signal
- Most non-recycled waste is **incinerated** at 4 waste-to-energy plants (not landfilled directly)

> **Site note:** Unlike many countries, Singapore incinerates ~90% of non-recycled waste. Only the ash + non-incinerable waste goes to Semakau. This is worth explaining — it's surprising and counterintuitive.

## 3. Semakau Landfill Facts

| Fact | Data |
|------|------|
| Location | 8km south of mainland Singapore |
| Total area | 3.5 km² |
| Total capacity | 63 million m³ |
| Current fill status | More than **50% full** |
| Projected full capacity | **2035** (at current disposal rates) |
| Years remaining (from Apr 2026) | **~9 years** |
| What goes there | Incineration ash + non-incinerable waste (not raw trash) |
| What it replaced | Singapore used to dump ash in the sea. Semakau was built on reclaimed land in 1999. |

**The punch line for the site:** Singapore has ONE landfill. It opened in 1999. It will be full by 2035. There is no Plan B site — Singapore is an island city-state with no space for another one.

> Source: [MSE Parliamentary Q&A on Semakau lifespan](https://www.mse.gov.sg/latest-news/written-reply-to-pq-on-extension-of-semakau-landfill-lifespan/), [NEA Semakau 20th Anniversary](https://www.nea.gov.sg/corporate-functions/resources/publications/books-journals-and-magazines/envision-lite/june-july-2020/semakau-landfill-20th-anniversary)

## 4. Product Life Stories — Key Data

### 🥡 Hawker Styrofoam Clamshell Box
- **Material:** Expanded Polystyrene (EPS) / Styrofoam
- **Decomposition time:** 500–1,000 years (or effectively never — it photodegrades into microplastics, not true decomposition)
- **Recyclable in Singapore?** No — not accepted in blue bins. NEA does not collect EPS curbside.
- **Fate:** Incinerated → ash → Semakau
- **Scale stat idea:** Singapore generates ~665,000 tonnes of plastic waste/year. Hawker centres serve ~2.8M meals/day.
- **Surprise fact:** The styrofoam doesn't go to Semakau as a box — it gets incinerated first. But the ash does.

### 🧋 Bubble Tea Cup
- **Material:** Most cups = #5 Polypropylene (PP) plastic. Lids = #6 Polystyrene. Straws = PP or biodegradable PLA. Sealing film = varies.
- **Decomposition time:** Plastic cup = 20–30 years. Lid = 500+ years. Straw = 200 years.
- **Recyclable in Singapore?** Cup (without residue) = technically yes (#5 PP). But contaminated with liquid/tapioca = no. Most end up in general waste.
- **Fate:** Incinerated → ash → Semakau
- **Scale stat idea:** Singapore has ~2,000+ bubble tea outlets. If each serves 100 cups/day = 200,000 cups/day minimum.

### 📦 Shopee / E-commerce Cardboard Box
- **Material:** Corrugated cardboard
- **Decomposition time:** 2 months in ideal conditions. In compacted landfill conditions: years (no light, water, or oxygen)
- **Recyclable in Singapore?** Yes — accepted in blue bins (must be clean and dry). Also collected by karung guni (rag-and-bone men).
- **Fate (if not recycled):** Incinerated → ash → Semakau
- **Good news angle:** Cardboard is one of the few items Singaporeans CAN meaningfully recycle — and karung guni culture makes it culturally familiar. The problem is contamination (wet/food-stained boxes).
- **Scale stat:** Singapore's e-commerce market grew significantly during COVID. Millions of deliveries per month.

### 🛍️ NTUC / Supermarket Plastic Bag
- **Material:** High-Density Polyethylene (HDPE) — #2 plastic
- **Decomposition time:** 10–20 years officially, but effectively never fully breaks down — photodegrades into microplastics
- **Recyclable in Singapore?** Technically yes (#2 HDPE), but soft plastics clog sorting machines — most facilities reject them. Not suitable for blue bins.
- **Fate:** Incinerated → ash → Semakau
- **2023 update:** Singapore introduced a mandatory charge for plastic bags at supermarkets in July 2023 (under the Resource Sustainability Act). S$0.05–$0.10 per bag. Early data shows reduced bag usage at major supermarkets.
- **Surprise:** Singapore charges for plastic bags at supermarkets now — but NOT at wet markets and mama shops.

### 📱 Old Smartphone (E-Waste)
- **Material:** Contains lead, mercury, cadmium, lithium, gold, copper, rare earth metals
- **Decomposition time:** Components take 1,000,000+ years. Toxic leaching begins in years.
- **Recyclable in Singapore?** YES — Singapore has one of Asia's most structured e-waste systems. StarHub, M1, Singtel, Harvey Norman, Courts, and NEA e-waste bins all accept old phones.
- **Fate if not recycled:** Incinerated — but toxic metals survive incineration and concentrate in ash → Semakau → potential groundwater contamination
- **Good news angle:** E-waste EPR framework launched 2021. Formal collection and certified recyclers required by law.
- **Scale stat:** Singaporeans are among the highest smartphone users in Asia. Average replacement cycle: ~2–3 years.

## 5. Government Policy Timeline (10 Years)

### 2015
- NEA launches **Food Waste Reduction** publicity programme — first major push on food waste awareness
- Singapore's overall recycling rate: **61%**

### 2016–2017
- NEA begins data collection and consultation for future EPR frameworks
- Recycling rate stable at 61%

### 2018
- Recycling rate peaks at **62%** — highest in the decade
- Government signals upcoming Zero Waste Masterplan

### 2019 ⭐ Major Year
- **Zero Waste Masterplan** released — Singapore's first comprehensive zero-waste strategy. Targets: cut waste to landfill by 30% by 2030, raise domestic recycling rate to 30%.
- **Resource Sustainability Act** enacted (October 2019) — legal foundation for EPR frameworks for e-waste, packaging, and food waste
- Recycling rate: 59%

### 2020
- COVID-19 pandemic — recycling rate drops to **52%** as collection disrupted and industrial activity drops
- NEA launches Bloobox (later relaunched) for household chemical waste collection

### 2021 ⭐
- **E-waste EPR framework** comes into effect — producers legally required to collect and recycle e-waste via licensed channels. Singapore becomes one of first SEA countries with mandatory e-waste EPR.
- NEA launches **Recycle Right** campaign to address blue bin contamination
- Recycling rate recovers to **55%**

### 2022
- **Mandatory Packaging Reporting (MPR)** expanded — more companies required to report their packaging data to NEA
- Singapore's first food waste segregation requirements piloted at selected premises
- Recycling rate: **57%**

### 2023 ⭐
- **Plastic bag charge** at large supermarkets (NTUC FairPrice, Cold Storage, Giant, Sheng Siong) introduced — S$0.05–$0.10 per plastic bag from July 2023
- **Beverage Container Return Scheme (BCRS) bill** amendment passed — legal basis for deposit return system established
- **Food waste segregation** made mandatory at large commercial premises (hotels, malls, convention centres)
- Recycling rate drops to **52%** (decline in C&D and slag waste)

### 2024
- Domestic recycling rate hits new low: **11%**
- Plastic recycling rate drops to **5%**
- Industry experts question feasibility of 30% domestic recycling target by 2030
- BCRS producer registration begins (August 2024)
- Deposit mark artwork guidelines published (November 2024)

### 2025
- BCRS product registration opens (October 2025)
- EPR framework for **packaging waste (plastics)** formally takes effect
- Singapore's waste generation rate continues downward trend (positive)

### 🔴 2026 — LAST 6 MONTHS (October 2025 – April 2026)

| Date | Milestone |
|------|-----------|
| Oct 2025 | BCRS product registration opens — beverage producers begin registering products |
| Nov 2024–Jan 2026 | Industry pushback and delays — producers complain about shifting deadlines, barcode rules, penalty uncertainty |
| **1 April 2026** | 🎉 **Beverage Container Return Scheme (BCRS) officially launches** — Singapore's first deposit return system. S$0.10 refundable deposit on all pre-packaged drinks in plastic bottles and metal cans (150ml–3L). Return machines ("reverse vending machines") deployed at selected locations. Target: 60% return rate in 2026, 80% by 2029. |
| By 30 Sept 2026 | Transition period: companies can still sell non-BCRS-marked beverages until this date |

> **BCRS is your "recent spotlight" moment** — it literally launched April 1, 2026. Three weeks ago. This is the most current, concrete thing the Singapore government has done on waste.

## 7. What Singaporeans Can Actually Do Right Now

*(For the footer section — practical, not preachy)*

1. **Blue recycling bins** — Paper, cardboard (clean/dry), glass bottles, metal cans, plastic bottles (#1 and #2). Do NOT put food-contaminated items, soft plastics, or styrofoam.

2. **E-waste drop-off** — StarHub, M1, and Singtel stores accept old phones/chargers. Harvey Norman, Courts, and IKEA accept larger electronics. NEA e-waste bins at community centres and selected locations.

3. **BCRS / Reverse Vending Machines (from April 2026)** — Return plastic bottles and cans to get your $0.10 deposit back. Locations to be expanded across Singapore.

4. **Karung guni / cardboard collectors** — The old-school Singaporean way. Still active. Leave bundled cardboard and old newspapers out for collection.

5. **Bloobox** — For household hazardous waste (paint, batteries, fluorescent tubes). Collection points at community clubs.

## 9. Supabase Architecture (If Needed)

> Anton mentioned having a Supabase account. Here's when to use it and when not to.

### Recommendation for v1: Don't use Supabase yet

For a static informational scroll site, all data above can live in **JSON files** or **markdown** in the repo. No backend needed. This keeps it:
- Free (no Supabase row limits)
- Fast (no API calls on page load)
- Simple (no auth, no rate limits)

### Use Supabase in v2 for:
- **Personal waste calculator** — store user sessions or aggregate data
- **Community sightings** — user-submitted photos + moderation
- **Admin CMS** — update government timeline entries without touching code
- **Analytics beyond GA** — custom event tracking

### If you want Supabase now (lightweight CMS approach):

```sql
-- Government timeline table
CREATE TABLE timeline_events (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  year int NOT NULL,
  month int,
  title text NOT NULL,
  description text NOT NULL,
  category text, -- 'policy', 'milestone', 'target', 'launch'
  is_recent boolean DEFAULT false, -- marks last-6-months events
  source_url text,
  created_at timestamptz DEFAULT now()
);

-- Products table
CREATE TABLE products (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  name text NOT NULL,
  emoji text,
  material text,
  decomposition_years int,
  decomposition_label text, -- e.g. "500–1,000 years"
  recyclable_in_sg boolean,
  blue_bin_accepted boolean,
  fate_in_sg text,
  fun_fact text,
  scale_stat text,
  display_order int
);
```

This would let you edit product data and timeline entries from Supabase dashboard without redeploying. Useful if you plan to keep the site updated.
